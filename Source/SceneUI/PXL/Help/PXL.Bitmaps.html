Types

TBitmapStorage = (...);

Type of storage used by bitmap.

Values
  • System: Image is currently stored in system memory, typically TPixelSurface.
  • Lockable: Image is currently stored in lockable texture, typically TCustomLockableTexture.
  • Drawable: Image is currently stored in drawable texture, typically TCustomDrawableTexture.
