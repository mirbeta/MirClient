Advanced Encryption Standard (AES), Delphi implementation

							Introduction

AES  is  an  standard  for  encrypting  information using secret key. It
supports  128-, 192- and 256-bit length keys. This Delphi implementation
is  based on FIPS draft standard and has been tested on Delphi 4, Delphi
5, C++Builder 5 and Kylix 1.

							Usage
							
Please see the enclosed documentation (ElAES.txt file) for information 
about routines of ElAES unit.
