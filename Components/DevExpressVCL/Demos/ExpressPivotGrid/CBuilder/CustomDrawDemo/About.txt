 In this demo you can:
  - Activate custom painting (by using the Options/Drawing menu) for the following pivot grid elements:
      data cells (including limit values)
      field headers
      column and row field values
      the pivot grid's background
  - Export data in either Excel format, as text, html or xml (the File/Export menu)
  - Change the location of grand totals (the Options/Totals Location menu)
  - Hide subtotals and grand totals (the Options/Totals Visibility menu)
  - Select a particular look and feel from the five look and feel options available (the Look&Feel menu)
